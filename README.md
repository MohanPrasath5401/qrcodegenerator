# QR Code Generator

This repo contains a Python script for generating QR codes locally and a web version hosted on GitHub Pages.

## Python Install
Please don't forget to install qrcode library
Command : pip install qrcode 
Run this command in your cmd

## To Save the QR 
In order to save the particular save it from different name after QR code is generated

## Python Version
Run `generate_qr.py` to create a QR code image from a URL.


## Web Version
Visit the GitHub Pages site to generate QR codes in your browser.

Hosted at: https://github.com/MohanPrasath5401/qrcodegenerator.
